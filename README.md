[한국 러스트 사용자 그룹 홈페이지][rust-kr] 소스코드
--------

### 사용법
```bash
git clone https://github.com/rust-kr/rust-kr.org.wiki.git docs --depth=1
cargo run
# 웹 브라우저에서 http://localhost:8000 를 열어보세요
```

###### Production usage
```bash
RUST_KR=production cargo run --release
```
